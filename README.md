# 🎧 Free Gym Music App by Jakub Kasprzak

Minimalistyczna aplikacja do odtwarzania muzyki inspirowana estetyką Apple Music.


## 🛠️ Tech Stack

- React Native (Expo)
- Expo Router
- NativeWind
- Expo AV / Audio API

## 📱 UI

Czysty, nowoczesny i minimalistyczny interfejs, inspirowany
Apple Music, wiele intuicyjnych rozwiązań

## 🚀 Jak zacząć?

```bash
npm install
npx expo start
